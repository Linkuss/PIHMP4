=============================PIHMP4 PROJECT============================



Team d'auteurs : Les croustillants

Membres : Jon Snow, Tiag, Luis (il sait), Fabien


Nom du projet : PIHMP4 (Projet Interface Homme-Machine : Puissance 4)


=======================================================================




Cette application est un jeu de puissance 4 que nous avons fait pour notre projet d'ihm.


Pour la lancer il fait absolument que le dossier lib soit dans le même dossier que l'executable.

Les langues supportées dans l'application pour l'instant sont : Français, Anglais, Allemand.

Plusieurs thêmes sont disponibles avec leur propres images et musiques !
